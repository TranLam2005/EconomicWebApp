Luồng xử lý theo mô hình phân lớp

View Layer

Thymeleaf admin pages: dashboard, catalog, orders, payments, discounts, users, blogs.

Controller Layer

AdminBackendController nhận request từ trang admin.

Service Layer

CategoryService, BrandService, ProductService, OrderService... xử lý nghiệp vụ.

Repository Layer

Repository thao tác với MySQL bằng Spring Data JPA.

Database Layer

MySQL lưu dữ liệu người dùng, sản phẩm, đơn hàng, thanh toán, blog...
